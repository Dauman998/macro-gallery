## 🌐 Deploy to GitHub Pages (Make It LIVE)

Want anyone to visit your site? Deploy to GitHub Pages (free forever!)

### Step-by-Step:

**1. Go to Settings**
- Visit: https://github.com/Dauman998/macro-gallery/settings/pages

**2. Configure Source**
- Under "Branch", click dropdown
- Select `copilot/create-photography-portfolio` (or merge to main first)
- Click "Save"

**3. Wait 1-2 Minutes**
- GitHub builds your site
- Refresh the page
- You'll see a green box with your URL

**4. Your Site is LIVE!**
- URL: `https://Dauman998.github.io/macro-gallery/`
- Share with anyone!
- Works on all devices
- Free forever

### Optional: Merge to Main First

**Why merge?**
- Cleaner repository
- Standard practice
- Main branch = published version

**How to merge:**
1. Go to: https://github.com/Dauman998/macro-gallery/pulls
2. Click on the pull request
3. Click "Merge pull request"
4. Confirm
5. Then enable GitHub Pages on `main` branch

## 🌐 Connect Custom Domain (Optional)

Want to use `danbaumanphotography.com` instead of GitHub URL?

### After Enabling GitHub Pages:

**1. Add Custom Domain in GitHub**
- Settings → Pages
- Scroll to "Custom domain"
- Enter: `danbaumanphotography.com`
- Click "Save"

**2. Update DNS Settings**
- See `HYBRID-WIX-GITHUB.md` for complete instructions
- If domain is on Wix, update DNS records there
- Point to GitHub Pages IP addresses

**3. Wait for DNS Propagation**
- Takes 1-24 hours (usually 2-4 hours)
- GitHub verifies domain
- Site accessible at custom domain!

## ❓ Common Questions

**Q: Do I HAVE TO deploy? Can I just keep it local?**
A: Yes! You can work locally forever. Deploy only when you want a public site.

**Q: If I deploy, can I still edit locally?**
A: Yes! Download files, edit, upload changes. Best of both worlds.

**Q: How do I update the live site?**
A: Edit files → Upload to GitHub → GitHub Pages updates automatically.

**Q: Is GitHub Pages really free?**
A: Yes! Free forever. No monthly fees, no credit card needed.

**Q: What if I want to delete the live site?**
A: Settings → Pages → Disable GitHub Pages. Site goes offline instantly.

**Q: Can I use both GitHub Pages AND my custom domain?**
A: Yes! Perfect combination. GitHub hosts free, domain makes it professional.

## 🆘 Troubleshooting

**"I clicked download but nothing happened"**
- Check Downloads folder
- Try different browser
- Use Method 1 (direct link)

**"ZIP file won't unzip"**
- Right-click ZIP → Open With → Archive Utility
- Or double-click ZIP file

**"Can't find index.html"**
- Look inside the unzipped folder
- It's at the top level
- Has .html extension

**"GitHub Pages not working"**
- Did you wait 2 minutes after enabling?
- Is the correct branch selected?
- Try Settings → Pages → Disable → Re-enable

**"Custom domain not working"**
- Wait 24 hours for DNS propagation
- Check DNS settings with dnschecker.org
- See HYBRID-WIX-GITHUB.md for help
