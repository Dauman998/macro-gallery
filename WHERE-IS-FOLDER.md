---

## 📥 Step 1: Download the Files First

**Use the direct download link:**

👉 **https://github.com/Dauman998/macro-gallery/archive/refs/heads/copilot/create-photography-portfolio.zip**

**Or:**
1. Go to: https://github.com/Dauman998/macro-gallery
2. Click "main" dropdown → Select "copilot/create-photography-portfolio"
3. Click green "Code" button → "Download ZIP"

---

## 📦 Step 2: Find the Downloaded ZIP

**Where it goes:**
- Downloaded ZIP goes to your **Downloads** folder automatically
- It's called: `macro-gallery-copilot-create-photography-portfolio.zip`

**How to find it:**
1. **Open Finder** (the smiling face icon in your Dock)
2. Click **"Downloads"** in the left sidebar
3. Look for the ZIP file (it's probably at the top, most recent)

**Or use Spotlight:**
1. Press **Cmd+Space** (command and spacebar together)
2. Type: `macro-gallery`
3. You'll see the ZIP file
4. Press Enter to open

---

## 📂 Step 3: Unzip the File

**Method 1: Double-click**
- Just double-click the ZIP file
- Mac automatically unzips it
- Creates a folder in the same location (Downloads)

**Method 2: Right-click**
- Right-click the ZIP file
- Select "Open"
- Folder appears in Downloads

**What you get:**
A folder named: `macro-gallery-copilot-create-photography-portfolio`

---

## 🏠 Step 4: Move Folder to Better Location

**The folder is in Downloads, but that's messy!**

**Move it to Desktop or Documents:**

1. Open Finder → Downloads
2. Find the folder: `macro-gallery-copilot-create-photography-portfolio`
3. **Drag it to Desktop** (or Documents folder)
4. Now it's easy to find!

**Optional: Rename it**
- Right-click the folder
- Select "Rename"
- Change to: `My Portfolio` or `Dan Bauman Photography`
- Easier to remember!
