## 🔗 Connect Your Domain to GitHub Pages

### Step 1: Add Domain to GitHub Pages

1. **Merge your Pull Request first** (if you haven't)
   - Go to: https://github.com/Dauman998/macro-gallery/pulls
   - Click your PR
   - Click "Merge pull request"

2. **Enable GitHub Pages**
   - Go to: https://github.com/Dauman998/macro-gallery/settings/pages
   - Under "Source": Select branch `main`
   - Click "Save"

3. **Add Custom Domain**
   - In the same GitHub Pages settings
   - Find "Custom domain" field
   - Type: `danbaumanphotography.com`
   - Click "Save"
   - Wait for DNS check...

### Step 2: Configure Your Domain Provider

**Where did you buy danbaumanphotography.com?**
- GoDaddy?
- Namecheap?
- Google Domains?
- Wix? (if it's currently on Wix)

**You need to update DNS settings there.**

#### For ANY Domain Provider:

**Add these DNS records:**

```
Type: A Record
Name/Host: @ (or leave blank)
Value: 185.199.108.153
TTL: Automatic (or 3600)

Type: A Record
Name/Host: @ (or leave blank)
Value: 185.199.109.153
TTL: Automatic

Type: A Record
Name/Host: @ (or leave blank)
Value: 185.199.110.153
TTL: Automatic

Type: A Record
Name/Host: @ (or leave blank)
Value: 185.199.111.153
TTL: Automatic

Type: CNAME Record
Name/Host: www
Value: Dauman998.github.io
TTL: Automatic
```

**Delete any old A records** pointing to other IPs (like Wix servers).

### Step 3: Wait for DNS to Update

- DNS changes take **1-48 hours** (usually ~2 hours)
- GitHub will check if it's configured correctly
- You'll see a green checkmark when ready
- Your site will work at `danbaumanphotography.com`!

## 📝 Provider-Specific Instructions

### If Your Domain is on GoDaddy:

1. Log in to GoDaddy
2. Go to "My Products"
3. Find `danbaumanphotography.com`
4. Click "DNS" button
5. Scroll to "Records" section
6. **Delete** old A records (if any)
7. Click "Add" for each new A record above
8. Click "Add" for CNAME record
9. Save changes

### If Your Domain is on Namecheap:

1. Log in to Namecheap
2. Click "Domain List"
3. Click "Manage" next to danbaumanphotography.com
4. Click "Advanced DNS" tab
5. Delete old A/CNAME records
6. Click "Add New Record" for each one above
7. Save changes

### If Your Domain is on Google Domains:

1. Log in to Google Domains
2. Click on danbaumanphotography.com
3. Click "DNS" on the left
4. Scroll to "Custom records"
5. Delete old records
6. Add new A and CNAME records
7. Save

### If Your Domain is on Wix:

**Problem:** Wix doesn't let you fully control DNS if you bought it through them.

**Solutions:**
1. **Transfer domain OUT of Wix** (to Namecheap, Google Domains)
2. **Or keep domain at Wix** and use their DNS settings (trickier)

**Want to transfer?** I can help you with that process.
