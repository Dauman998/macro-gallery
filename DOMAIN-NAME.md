## 🛠️ How to Set Up Custom Domain (When Ready)

### Step 1: Buy Your Domain
1. Go to Namecheap.com
2. Search for domain (e.g., "danbauman")
3. Add `.com` or `.photography` to cart
4. Check out (~$12)
5. You own the domain!

### Step 2: Point Domain to GitHub Pages
**In Namecheap (or your domain provider):**
1. Log in
2. Go to "Domain List"
3. Click "Manage" next to your domain
4. Click "Advanced DNS"
5. Add these records:

```
Type: A Record
Host: @
Value: 185.199.108.153

Type: A Record
Host: @
Value: 185.199.109.153

Type: A Record
Host: @
Value: 185.199.110.153

Type: A Record
Host: @
Value: 185.199.111.153

Type: CNAME Record
Host: www
Value: Dauman998.github.io
```

### Step 3: Configure GitHub Pages
1. Go to your repo: https://github.com/Dauman998/macro-gallery
2. Click Settings
3. Click Pages (left sidebar)
4. In "Custom domain" field, enter: `yourdomain.com`
5. Click Save
6. Wait 24 hours for DNS to update

### Step 4: Done!
- Your site is now at `yourdomain.com`
- GitHub Pages still hosts it (free!)
- You just own the domain name ($12/year)

## ❓ Common Questions

**Q: Do I NEED a custom domain?**
A: No! GitHub Pages URL works great. Custom domain is optional branding.

**Q: Can I change my GitHub Pages URL?**
A: Only by renaming your repository. But easier to add custom domain.

**Q: What if my name is taken?**
A: Try variations:
- `yourname.photography`
- `yournamephoto.com`
- `yourname.co`
- `yourname.art`

**Q: Can I buy domain first, deploy later?**
A: Yes, but better to deploy first (test free), then add domain.

**Q: Does custom domain cost extra hosting?**
A: No! GitHub Pages hosting stays FREE. You just pay for the domain name ($12/year).

**Q: Will my site break if I add a domain later?**
A: No! Your GitHub URL keeps working. Custom domain just points to the same site.
