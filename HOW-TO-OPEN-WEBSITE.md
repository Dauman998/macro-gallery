# 🌐 How to Open Your Website

## ✅ The Right Way:

**Double-click these files to view your website:**
- `index.html` - Home page (start here!)
- `gallery.html` - Portfolio page
- `about.html` - About page
- `contact.html` - Contact page

---

## ❌ Don't Click These:

**These are code files - they will show as text if opened directly:**
- `navigation.js` - Menu JavaScript code
- `lightbox.js` - Lightbox JavaScript code
- `scroll-effects.js` - Scroll animations code
- `styles.css` - CSS styling code

---

## 🎯 What You'll See:

### If You Open HTML Files (✅ Correct):
- Beautiful website
- Working menu
- Clickable photos
- Everything functions properly

### If You Open JS/CSS Files (❌ Wrong):
- Code displayed as text
- HTML entities like `&amp;` and `&gt;`
- Nothing works
- Confusing!

---

## 📋 Quick Guide:

### To View Your Website:
1. Go to your website folder
2. **Double-click `index.html`**
3. Website opens in browser
4. Enjoy!

### To Edit Code:
1. **Right-click** the file
2. Choose "Open With"
3. Select TextEdit or VS Code
4. Edit and save
5. Refresh browser to see changes

---

## 🔧 Troubleshooting:

### "I see JavaScript code instead of website"
**Problem:** You clicked a `.js` file
**Solution:** Close it and open an `.html` file instead

### "Nothing is working"
**Problem:** You might be viewing a code file
**Solution:** Make sure the URL ends with `.html`

### "I see weird symbols like &amp;"
**Problem:** Browser is displaying code as text
**Solution:** Open the `.html` file, not `.js` file
