# Where to Comment on GitHub

You asked: "Where is comment? The little paper airplane at the bottom right?"

Let me explain GitHub's interface!

## 🤔 The Paper Airplane Icon

**That's NOT for comments!**

The paper airplane icon (bottom-right) is **GitHub Copilot Chat** - that's where you're talking to me right now! It's an AI assistant for asking questions.

## 💬 Where to Add Comments (For File Uploads)

**On a Pull Request or Issue:**

1. Scroll **all the way to the bottom** of the page
2. You'll see a large text box with "Write" and "Preview" tabs
3. Above it says "Leave a comment"
4. **That's the comment box!**

**To upload a file there:**
- Drag your screenshot into that big text box
- OR click the image icon in the toolbar
- OR paste directly (Cmd+V)

## 🗂️ GitHub Interface Areas

**Different places in GitHub:**

### 1. Copilot Chat (Paper Airplane) ✈️
- Bottom-right corner
- For asking AI questions
- Where you're talking to me now
- **Can't upload files here easily**

### 2. Comment Box 💬
- Bottom of PR/Issue page
- Scroll way down
- Large text area
- **Can upload files here**

### 3. Files Tab 📁
- In the repository
- Can upload files to repo
- Different from comments
- For adding website files
