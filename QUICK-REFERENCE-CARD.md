## 📥 Download Updated Files

**Simple Steps:**
1. Go to **GitHub.com**
2. Search for **"macro-gallery"**
3. Click the green **"Code"** button
4. Click **"Download ZIP"**
5. Unzip the file
6. Replace your old folder with the new one

**Remember:** Green button → Download ZIP → Unzip → Replace

## 📤 Upload Your Local Changes

**Easiest Method:**
1. Go to **GitHub.com/Dauman998/macro-gallery**
2. Click **"Upload files"** button (top right area)
3. Drag your edited files from your Mac
4. Write a short description (e.g., "Updated photos")
5. Click **"Commit changes"**

**Remember:** Upload button → Drag files → Commit

**Alternative (for single file):**
1. Navigate to the file on GitHub
2. Click the pencil icon (Edit)
3. Make your changes
4. Click "Commit changes"

## 🎯 Complete Workflow

**To Update Your Live Website:**

1. **Download** newest files (green Code button → Download ZIP)
2. **Edit** files on your Mac (add photos, change text)
3. **Preview** locally (double-click index.html)
4. **Test** all animations (they all work!)
5. **Upload** to GitHub (Upload files button → drag & drop)
6. **Wait** 2 minutes for GitHub Pages to update
7. **Visit** your live site to see changes
8. **Done!**
