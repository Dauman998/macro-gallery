## 💻 How to Download Everything to Your Mac

### Method 1: Download ZIP (Easiest!)

**Step-by-step:**

1. **Open Safari or Chrome**

2. **Go to your repository:**
   ```
   https://github.com/Dauman998/macro-gallery
   ```

3. **Look for the green button** that says **"Code"**
   - It's on the right side, above the file list
   - Green color, can't miss it!

4. **Click the "Code" button**
   - A menu drops down

5. **Click "Download ZIP"**
   - At the bottom of that menu
   - Your Mac will download a file called `macro-gallery-main.zip`

6. **Find the downloaded file:**
   - Usually in your Downloads folder
   - Double-click it to unzip

7. **Move to a good location:**
   - Drag the `macro-gallery-main` folder to your Desktop
   - Or Documents folder
   - Somewhere you can find it!

8. **Done!** 
   - Open the folder
   - You have EVERYTHING!

### Method 2: GitHub Desktop (For Future Updates)

**One-time setup (5 minutes):**

1. **Download GitHub Desktop:**
   - Go to: https://desktop.github.com
   - Download for Mac
   - Install it (drag to Applications)

2. **Sign in:**
   - Open GitHub Desktop
   - Sign in with your GitHub account

3. **Clone your repository:**
   - Click "File" → "Clone Repository"
   - Find "Dauman998/macro-gallery"
   - Choose where to save (like Desktop)
   - Click "Clone"

4. **Done!**
   - Folder appears on your Mac
   - All files inside!

**Advantage:**
- When you update files on GitHub
- Click "Fetch origin" in GitHub Desktop
- Get latest version automatically!
