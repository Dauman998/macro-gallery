## 🎨 Common Vertical Line Patterns

Here are 6 common ways vertical lines are used in photography portfolio headers:

### Option 1: Left Border Accent (Most Common)

```
│ dan bauman
│ photography
```

**What it looks like:**
- Vertical line on the left side
- Spans both lines of text
- Creates a clean, minimal accent
- Very popular in modern portfolios

---

### Option 2: Left with Offset

```
│   dan bauman
    photography
```

**What it looks like:**
- Vertical line on far left
- Name indented/offset from line
- Creates visual hierarchy
- Artistic, editorial feel

---

### Option 3: Tall Decorative Line

```
    │
    │
│ dan bauman
│ photography
    │
    │
```

**What it looks like:**
- Vertical line extends beyond text
- Taller than the header content
- Strong design element
- Bold, architectural look

---

### Option 4: Between Lines

```
dan bauman
    │
photography
```

**What it looks like:**
- Separator between name and tagline
- Creates clear division
- Can be centered or offset
- Classic, traditional feel

---

### Option 5: Right Side Accent

```
dan bauman │
photography │
```

**What it looks like:**
- Vertical line on right side
- Mirrors left-accent style
- Less common but unique
- Modern, unexpected placement

---

### Option 6: Inline Separator

```
dan bauman │ photography
```

**What it looks like:**
- Single line with vertical bar between
- Compact design
- Traditional separator style
- Space-efficient

## 📤 Alternative Image Upload Sites

If you want to upload your screenshot, try these free sites:

### 1. Imgur.com ⭐ RECOMMENDED

**Why:** No account needed, super fast, works great

**How:**
1. Go to https://imgur.com
2. Click "New post" button (or just drag image onto page)
3. Drag your screenshot onto the page
4. Wait for upload (few seconds)
5. Right-click the image → "Copy image address"
6. Paste link here!

### 2. Postimages.org

**How:**
1. Go to https://postimages.org
2. Click "Choose images"
3. Select your screenshot
4. Click "Upload"
5. Copy the "Direct link"
6. Paste link here!

### 3. ImgBB.com

**How:**
1. Go to https://imgbb.com
2. Click "Start uploading"
3. Select your screenshot
4. Wait for upload
5. Copy the link
6. Paste link here!

### 4. Google Drive

**How:**
1. Go to https://drive.google.com
2. Click "New" → "File upload"
3. Select your screenshot
4. Right-click file → "Share" → "Anyone with the link"
5. Copy link
6. Paste link here!

### 5. Dropbox

**How:**
1. Upload to Dropbox
2. Click "Share" button
3. Click "Create link"
4. Copy link
5. Paste link here!
